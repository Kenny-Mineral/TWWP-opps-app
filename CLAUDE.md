# CLAUDE.md — Standing rules for this repo

## Before every commit

After making any code changes and before committing, always:

1. Update `docs/CURRENT_STATE.md` to reflect what changed
2. Mark any resolved backlog items in `docs/backlog.md` as ✅
3. Update `docs/known-issues.md` if any bugs were fixed

Include the doc updates in the same commit as the code changes.

## At the start of each new session

Ask the user whether the doc-update rule above is still needed for this session.
